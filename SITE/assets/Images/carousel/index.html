Image tips:

  • Use high-quality images with a consistent aspect ratio (16:9 recommended)
  • Optimize images for web (file size under 500KB if possible)
  • Recommended dimensions: 1200-1800px width
  • Use JPG format for photos, PNG for graphics with transparency
